Tools used to make this clone.
=============================

 * Corona SDK (http://coronalabs.com/) - SDK used to write the game.
 * bfxr (http://www.bfxr.net/) - Used to generate basic sounds.
 * Audacity (http://audacity.sourceforge.net/) - Sound editor used to modify sound files for our game.
 * Sublime (http://www.sublimetext.com/) - Code editor.
